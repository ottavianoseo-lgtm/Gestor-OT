# 🏛️ ARQUITECTURA ENTERPRISE: MULTI-TENANCY & MAGIC LINKS

## 1. ARQUITECTURA MULTI-TENANT (AISLAMIENTO LÓGICO)
El sistema debe soportar múltiples empresas bajo una misma base de datos, con aislamiento total de datos.

### 1.1 Modelo de Datos (ITenantEntity)
- **Interfaz:** Todas las entidades Core (`WorkOrder`, `Labor`, `Campaign`, `CampaignLot`, `CropStrategy`) deben implementar `ITenantEntity`.
- **Propiedad:** `Guid TenantId { get; set; }`.
- **Global Query Filters (EF Core 10):**
  - En `OnModelCreating`, configurar el filtro automático:
    `modelBuilder.Entity<T>().HasQueryFilter(e => e.TenantId == _currentTenantService.TenantId);`
  - Esto garantiza que un ingeniero NUNCA vea datos de la empresa A mientras trabaja en la empresa B.

### 1.2 Flujo de Autenticación & Contexto
1.  **Login:** El usuario se autentica (Supabase Auth).
2.  **Selector de Contexto:** Si el usuario tiene acceso a >1 Tenant, el sistema muestra una pantalla intermedia: *"Seleccionar Empresa"*.
3.  **Inyección de Claim:** Al seleccionar, se guarda el `TenantId` en el estado de la aplicación (o en un Header HTTP `X-Tenant-ID` para cada petición a la API).

## 2. SISTEMA "MAGIC LINKS" (ACCESO SIN CUENTA)
Permite a terceros (Contratistas) visualizar y operar una OT específica sin usuario ni contraseña.

### 2.1 Generación de Token (Backend)
- **Endpoint:** `POST /api/workorders/{id}/share`.
- **Lógica:**
  1.  Genera un `Guid` criptográficamente seguro o un JWT firmado.
  2.  Guarda el hash del token en tabla `SharedAccessTokens` con `ExpirationDate` (ej: 7 días).
  3.  Retorna la URL: `https://app.gestorot.com/public/ot/{token}`.

### 2.2 Visor Público (Frontend - Blazor WASM)
- **Layout:** `PublicLayout.razor` (Sin Sidebar, sin Menú, ultra-ligero).
- **Página:** `PublicWorkOrder.razor`.
- **Seguridad:**
  - Esta ruta permite acceso anónimo (`AllowAnonymous`).
  - Al cargar, valida el token contra la API.
  - **Permisos:** Solo permite ver la OT y cambiar el estado de las labores a `Realized` (Checklist de contratista). No permite ver precios ni editar estrategias.
